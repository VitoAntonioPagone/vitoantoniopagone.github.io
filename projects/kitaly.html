Kitaly – Modern Web Platform

→ Visit the Website

Full-stack web project with strong focus on frontend, backend, and database design.

Overview

Kitaly is a modern web platform designed to feel fast, intuitive, and dependable from the very first click. The project balances an elegant user experience with a reliable technical foundation, keeping the interface clean while ensuring the system behind it is solid and scalable.

It was a real pleasure to work on this project — I’m a huge fan of football, and building something connected to that world made the process even more enjoyable.

Architecture & Technology

The frontend focuses on clarity and flow, ensuring that content is easy to access and interactions feel natural. The visual layout is simple, modern, and responsive, designed to adapt cleanly from desktop to mobile without losing structure.

The backend and database layer provide the reliability needed to keep everything consistent as the platform grows. The core stack is centered on Flask with SQLAlchemy and a MySQL database, paired with Jinja2 templates, Tailwind CSS, and vanilla JavaScript for a responsive, fast frontend. In production it runs behind Gunicorn and Nginx, keeping the platform stable and scalable without drawing attention away from the experience itself.

In short, Kitaly is a full-stack build where design and engineering move together — a modern web experience supported by a solid technical core.
